# Mongeral Produt-Front #

### Features:

1. Create an product.
2. Retrieve product list.
3. Retrieve category list.
4. Delete an Product.
5. Search product using filters. 

### Install:

1. clone git the final test repository.
2. npm install.
3. ng serve.
4. for production(generate dist directory): ng build.
5. for tests: ng test.
